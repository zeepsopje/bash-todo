# bash-todo
Simple todo application written in bash that I use for my personal projects.

## Installing
```
git clone git@github.com:zeepsopje/bash-todo.git &&
cd bash-todo &&
sudo make install
```

## Usage
### Show tasks
```
todo
```
or

```
todo list
```

### Add a task
```
todo add My task
```

### Delete a task
```
todo done [index]
```

### Delete all tasks
```
todo clear
```
