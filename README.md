# bash-todo
Simple todo application written in bash that I use for my personal projects.

## Installing
```bash
git clone git@github.com:zeesopje/bash-todo.git
cd bash-todo
sudo make install
```

## Usage
### Show tasks
```bash
todo
```
or

```bash
todo list
```

### Add a task
```bash
todo add My task
```

### Delete a task
```bash
todo delete [index]
```

### Delete all tasks
```bash
todo clear
```
